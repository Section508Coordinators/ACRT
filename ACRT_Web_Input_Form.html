Refer to the ACRT User Guide  for complete instructions.

Tool Tips:

  1. Use 'Skip to main content' link or keyboard shortcut Alt+M to go to main content.
  2. To Create a new Report, Load a JSON Trusted Tester v.5.1.1 (for web) or Trusted Tester v.4 (for software) (TT5.1.1 or TT4) Success Criteria template.
  3. To edit, select JSON file with test results.
  4. Expand Product, Test Environment, and Tester Sections to enter information.
  5. Complete all required form fields marked with an asterisk (    ) in Product and Testing information sections.
  6. You can customize JSON Success Criteria templates with pre-populated information (e.g., Product Section) for future use.
  7. If needed, open a copy of the Trusted Tester Test Process.
  8. Trusted Testers should coordinate with their IT Project Manager regarding remediation details.
